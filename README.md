# Beng

<div align="center">
    ![](assets/beng.svg)
</div>

A preset for rollup.

## Motivation

I'm tired of configuring rollup for every project. So I made this preset.

## Usage

Optional, but recommended:

```ts
import { defineConfig } from '@akrc/beng'

export default defineConfig({});
```

```bash
pnpm beng build
```

Or you can create it with `create-beng`:

```bash
pnpm create beng
```
